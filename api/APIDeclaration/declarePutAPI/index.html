class APIDeclaration

method declarePutAPI

Purpose

Used for declaring an API using the HTTP PUT method

Usage

APIs can be declared at any time.

Parameters

NameTypeDescription
pathstringPath to the API under the base URL of the APIDeclaration.

Returns

IncompleteApiDeclaration

Examples

import {APIDeclaration} from 'papupata'
const api = new APIDeclaration()

const MyAPI = api.declarePutAPI('/api/person')
  .response<string>()
